# XTC-CANTEN
Artist Website Valhala Cult

## GitHub Pages Setup

This website is ready to run on GitHub Pages!

### To Enable GitHub Pages:

1. Go to your repository settings: `Settings > Pages`
2. Under "Build and deployment", select:
   - **Source**: `Deploy from a branch`
   - **Branch**: `main`
   - **Folder**: `/ (root)`
3. Click **Save**

Your site will be published at: `https://Hxnan25.github.io/XTC-CANTEN/`

The `index.html` file is configured with all URLs updated for the GitHub Pages domain.

### Files:
- `index.html` - Main website (configured for GitHub Pages)
- `xtc-canten.html` - Original version (for reference)
